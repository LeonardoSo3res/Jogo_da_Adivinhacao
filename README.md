# Jogo_da_Adivinhacao
Um programa em JS que peça ao usuário para adivinhar um número entre 1 e 10. Verificar se o usuário acertou o número gerado aleatoriamente. Permitir que o usuário continue tentando até acertar. Informar ao usuário se o número inserido é maior ou menor que o número correto. O jogador terá 3 chances para acertar o número;
